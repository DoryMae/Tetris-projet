Tetris est un jeu qui a pour but de mettre le joueur au défi de réaliser des lignes complètes en déplaçant des pièces de formes différentes, les tétrominos qui descendent du haut de la grille de jeu.
Les lignes complétées disparaissent tout en rapportant des points et le joueur peut de nouveau remplir les cases libérées. Le jeu n’a pas de fin : le joueur perd la partie lorsqu’un tétromino reste bloqué en haut. Une ligne complète rapporte 40 points, deux lignes complètes rapportent 100 points, trois lignes font 300 points tandis que le maximum est obtenu avec quatre lignes
complètes d’un coup qui rapportent 1200 points. 
Pour compiler le programme, veuillez entrer dans le terminal la commande suivante : make all.
Une fois la compilation terminé, veuillez saisir la commande suivante pour exécuter le programme : .\prog.exe

Une fois le programme lancé, les instructions à effectuer seront inscrits à l’écran .
Alors, il ne me reste qu’à vous dire bon jeu à vous.
